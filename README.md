# 🏆 2026 世界杯 · 大数据实时分析

## 📱 安装到手机

### iPhone / iPad (iOS Safari)

1. 用 **Safari** 打开 `index.html`
2. 点击底部 **分享按钮** (⬆)
3. 滑动找到 **「添加到主屏幕」**
4. 点击「添加」

> ✅ 之后会像原生 App 一样出现在桌面，没有浏览器边框

### Android (Chrome / Edge)

1. 用 **Chrome** 打开 `index.html`
2. 点击右上角 **⋮** → **「添加到主屏幕」** / **「安装应用」**
3. 确认安装

> 如果没弹出安装提示，在 Chrome 地址栏右侧会有安装图标

## 🧠 功能一览

| 功能 | 说明 |
|------|------|
| 📅 **实时赛程** | 进行中比赛高亮、今日赛程、近期赛果 |
| 📊 **实力分析** | ELO 动态评分、球队排名、近期状态、智能预测 |
| ⚔️ **交锋记录** | 任选两队查看历史世界杯交手记录 (2014-2022) |
| 📋 **积分榜** | 12组实时积分、进失球、晋级形势 |
| 🏆 **淘汰赛** | 32强 → 决赛完整对阵图 |

### 数据来源
- **实时比分**: TheSportsDB API (免费)
- **历史交锋**: 2014/2018/2022 世界杯官方数据
- **ELO 评分**: FIFA 排名基准 + 比赛结果动态计算
- **预测模型**: ELO 期望胜率公式
